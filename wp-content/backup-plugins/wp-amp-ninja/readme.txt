The plugin would help in adding support for Accelerated Mobile Pages (AMP) Project, and this is an open source program. 

== Description ==
WordPress users can now easily publish and also manage the content directly from their WordPress to Google Accelerated Mobile Pages (AMP) with the help of our plug-in WP AMP. The plug-in provides full support for the ads and the analytics. The WP AMP plug-in does not need any additional configuration, just install and activate and you are all done!

Basic Features:

- Automatically integrate AMP to the website, no coding require
- Web master can upload their own logo and favicon icon
- Plug-in supports page, post, custom post type, taxonomy, tags and archive pages
- Ability to display logo or blog name in the header
- Ability implement Google Analytics for visitor tracking
- Support of GIF image, Video, Iframe and Audio tags

Premium Features:

- Option to choose SCHEMA for each post type, supported Schemas are WebPage, BlogPosting and Article
- Show/hide featured image
- Select and show/hide categories/taxonomy, It will display for relevant post types only
- Show/hide recent posts and update section title
- Show/hide related posts and update section title
- Change footer copyright information
- Change Theme colors
- Header Menu for AMP Pages
- Ability to add Custom CSS
- Support of Typography
- Show/hide Social Media icons - Facebook, Twitter, Google+, LinkedIn, Pinterest, Email and Whatsapp
- Remove Open Graph meta from header section which is included by plugin automatically
- Show/hide multiple AdSense
- Rearrange position of AdSense
- Rearrange position of AdSense
- Disable Post Type or specific Post for AMP pages

Once the WP AMP plug-in is activated, the website dynamically generates the AMP version. It uses BlogPosting as default schema.

Accelerated Mobile Pages or AMP is an open source initiative which was developed with an intention to improve the mobile content ecosystem for creators, users, publishers and consumer platforms. 

How to Install?

From your WordPress dashboard, visit "Plugins". Then search for the "WP AMP" plug-in. Click on "Install Now". Once installed activate the WP AMP plug-in from the plug-ins page. 

Once the WP AMP plug-in is activated Visit WP AMP plug-in settings and follow the instructions to configure the plug-in.

Compatibility - it requires WP version of 3.0.1 or higher and compatible upto 4.6.


== Installation ==
1. Upload the folder to the /wp-content/plugins/ directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Set the logo from WP AMP Settings section in admin area.
4. You can access your AMP enabled website by adding ?wpamp at the end of the url.
Ex. http://{YOURDOMAIN}.com/{SLUG}?wpamp
